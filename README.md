# hosting_webapp_s3-githubactions
hosting webapp using AWS S3 and Github Actions  


### Create a repository and add the necessary files to it.
### Create an S3 bucket and configure the required policy.
### Create an IAM user, attach the S3 policy, and generate access key and secret access key ID.
### Add the access key and secret access key ID as secrets in the GitHub Actions repository.


### Write the main.yml file to define the GitHub Actions workflow.
